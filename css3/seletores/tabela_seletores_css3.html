Exemplo de Seletores CSS3

Seletor de Atributo

Exemplo E[att="val"]

  • Primeiro exemplo
    CSS
                                    div#atributo p[title]{color:red;}
    HTML
                                    
    
                                    p[title] - Elemento "P" com attributo "title"
    Resultado

    p[title] - Elemento "P" com attributo "title"

  • Segundo exemplo
    CSS
                                    /* @annotation: class especifica */
                                div#atributo p.dois {color:green;}
    HTML
                                    
    
                                    p.dois - Elemento "P" com classname igual "dois" sendo assim com estilo de cor Amarelo
    Resultado

    p.dois - Elemento "P" com classname igual "dois" sendo assim com estilo de cor ESPECIFICA

  • Terceiro exemplo
    CSS
                                    /* E[att="val"] */
                                div#atributo p[title="especifico"]{color:blue;}
    HTML
                                    
    
                                    p[title="especifico"]{color:blue;} - Elemento "P" com atributo "TITLE" com valor "especifico"
    RESULTADO

    p[title="especifico"]{color:blue;} - Elemento "P" com atributo "TITLE" com valor "especifico"

  • Quarto exemplo
    CSS
                                    /* @annotation:lang="pt" */
                                 div#atributo p[lang="pt"] {
                                    color:gray;
                                }
    HTML
                                    
    
                                    p[lang="pt"] - Elemento P cujo valor do atributo "lang" é "pt"
    RESULTADO

    p[lang="pt"] - Elemento P cujo valor do atributo "lang" é "pt"

Exemplo E[att^="val"] - Elemento E cujo valor do atributo "foo" começa com a string "bar"

    exemplo
    CSS
     

                            /* @annotation:a[rel$="imo"] - Valor do atributo "rel" termina com a string "imo" */
                             div#atributo a[rel$="imo"] {
                                color:gray;
                            }

                            /* @annotation:a[rel^="minimo"] - Valor do atributo "rel" começa com a string "minimo" */
                             div#atributo a[rel^="minimo"] {
                                color:green;
                            }
    HTML
                                Estilo aplicado(termina com imo sendo assim cinza)
                            Estilo não aplicado
                            Estilo aplicado
    RESULTADO
    Estilo aplicado(termina com imo sendo assim cinza) Estilo não aplicado Estilo aplicado

E[foo$="bar"] - Elemento E cujo valor do atributo "foo" termina com a string "bar"

                            }

E[foo*="bar"] - Elemento E cujo valor do atibuto "foo" contém a substring "bar"

    exemplo
    CSS
     
                             div#atributo a[rel*="que"] {
                                color:red;
                            }
    HTML
                                queijo
                            qualquer
                            martelo
    RESULTADO
    queijo qualquer martelo
